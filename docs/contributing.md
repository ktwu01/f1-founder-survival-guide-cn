# 贡献指南

欢迎补充高质量、可验证、对中文 F1 创业者有帮助的内容。

**我们相信**：最好的生存手册是由正在经历这一切的人共同编写的。你的经验，无论成功还是失败，都能帮助下一个 F1 创业者少走弯路。

## 提交前检查

- 有明确来源链接（官方文档、可靠机构、原始文章）。
- 有中文摘要（3-8 行，说明适用人群与边界）。
- 有风险提示（哪些情况不适用，可能踩坑点）。

## 建议模板

1. 背景与问题
2. 适用人群
3. 核心结论（附来源）
4. 风险与例外
5. 可执行下一步

## 不接受内容

- 无来源结论
- 过度承诺（例如“100% 可行”）
- 未授权全文搬运
