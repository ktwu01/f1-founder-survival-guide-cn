# 华人创业成功案例（包括F1签证持有者）

## 前言
本文档收集100个华人真实的创业成功例子，特别关注那些从F1学生签证开始创业的案例。这些案例旨在为留学生和创业者提供参考和启发。

## 案例列表（当前收集：10个案例）

### 1. Eric Yuan（袁征）
- **背景**: 山东科技大学毕业，1997年持学生签证赴美
- **教育**: 斯坦福大学MBA
- **创业**: Zoom Video Communications创始人兼CEO
- **成就**: 2020年疫情期间Zoom成为全球最流行的视频会议平台，公司市值一度超过1000亿美元
- **签证路径**: F1学生签证 → H-1B工作签证 → 绿卡 → 公民
- **F1创业**: ✅ 是

### 2. Tony Xu（徐迅）
- **背景**: 南京出生，8岁随父母移民美国
- **教育**: 加州大学伯克利分校（本科），斯坦福大学商学院（MBA）
- **创业**: DoorDash联合创始人兼CEO
- **成就**: 2020年DoorDash上市，成为美国最大的外卖平台之一
- **备注**: 虽然不是直接从F1创业，但作为移民背景的成功案例

### 3. Min Zhu（朱敏）
- **背景**: 浙江大学本科，斯坦福大学博士
- **教育**: 斯坦福大学工程学博士

- **成就**: WebEx于2007年被思科以32亿美元收购
- **签证路径**: F1学生签证开始在美国学习

### 4. Kai-Fu Lee（李开复）

- **职业**: 微软中国研究院院长、Google中国总裁、创新工场创始人
- **创业**: 创新工场（中国领先的创业投资机构）
- **签证路径**: F1学生签证开始在美国学习

### 5. Jenny Lee（李宏玮）

## 收集标准

1. **华人背景**: 中国出生或华裔
2. **美国创业**: 主要创业活动发生在美国
3. **签证情况**: 优先收集从F1学生签证开始创业的案例
4. **成功定义**: 公司上市、被收购、达到显著规模或影响力
5. **可验证性**: 有公开报道和可靠信息来源
