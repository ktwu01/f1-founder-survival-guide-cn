# 03 失败与退出路径

目标：体面地退场，也是一种本事。别让创业失败变成人生的污点，而是一段"昂贵但牛逼"的经历。

## "我想再撑一个月"后的悲剧

- **"账上只剩 $5000 了，下个月要发 $10000 的工资"**  
  （恭喜你，你已经违法了。拖欠工资在美国是很严重的，劳工局分分钟教你做人。）
- **"F1 OPT 还有 60 天到期，公司黄了"**  
  （别慌，你是去申请 Day 1 CPT，还是赶紧找个婚结了？这是个战术问题。）
- **"投资人说再看看数据就给钱"**  
  （翻译：没戏。赶紧准备 B 计划，不管是去大厂搬砖还是回国接班。）
- **"我要把公司注销了，直接不管就行了吧？"**  
  （Delaware 州政府：你想得美。欠税罚款会一直累积，直到你再次踏入美国海关的那一刻。）

## 需要你的贡献

**这个部分需要你的帮助！** 如果你有创业失败、公司关闭、身份转换等方面的经验，欢迎贡献内容。

### 触发停止线（Stop-Loss）
- **现金流警戒线**：剩多少钱必须关门？如何计算最低生存资金？
- **心理防线**：如何识别 burnout 信号？什么时候应该寻求帮助？

### 退出前必须完成的事项清单
- **员工处理**：如何合法解雇员工？WARN Act 适用吗？
- **资产处理**：服务器、SaaS 订阅、办公室租约如何处理？
- **投资人沟通**：如何与投资人沟通失败？如何保护个人声誉？

### 失败复盘与再出发
- **简历处理**：如何将创业失败经历转化为求职优势？
- **身份转换**：F1/OPT 失败后有哪些合法身份转换路径？
- **心理恢复**：如何从创业失败中恢复，准备下一次尝试？

**欢迎提交 Pull Request 或通过 Issues 分享你的经验！** 你的失败经验可能正是别人需要的救命稻草。
