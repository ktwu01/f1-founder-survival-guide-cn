# 03 失败与退出路径

目标：体面地退场，也是一种本事。别让创业失败变成人生的污点，而是一段"昂贵但牛逼"的经历。

## "我想再撑一个月"后的悲剧

- **"账上只剩 $5000 了，下个月要发 $10000 的工资"**  
  （恭喜你，你已经违法了。拖欠工资在美国是很严重的，劳工局分分钟教你做人。）
- **"F1 OPT 还有 60 天到期，公司黄了"**  
  （别慌，你是去申请 Day 1 CPT，还是赶紧找个婚结了？这是个战术问题。）
- **"投资人说再看看数据就给钱"**  
  （翻译：没戏。赶紧准备 B 计划，不管是去大厂搬砖还是回国接班。）
- **"我要把公司注销了，直接不管就行了吧？"**  
  （Delaware 州政府：你想得美。欠税罚款会一直累积，直到你再次踏入美国海关的那一刻。）

## 触发停止线（Stop-Loss）—— 如何识别"该停了"

### 现金流警戒线
- **3个月法则**：如果公司现金流无法支撑未来3个月的运营成本，立即启动退出计划
- **工资覆盖**：确保有足够资金支付所有员工未来2个月的工资和福利
- **紧急储备金**：至少保留$10,000用于公司关闭的法律和行政费用

### 心理防线信号
- **持续失眠**：连续2周以上因公司问题无法正常睡眠
- **健康问题**：出现焦虑症、抑郁症症状，或已有健康问题恶化
- **关系破裂**：与合伙人、家人、朋友关系因公司压力严重受损
- **失去热情**：对产品或客户完全失去兴趣，每天只是"熬时间"

### 投资人信号
- **连续3次"再看看"**：投资人连续3次要求"更多数据"而不给明确承诺
- **估值大幅下调**：新投资人要求估值比上一轮低50%以上
- **条款恶化**：出现对赌协议、个人担保等高风险条款

### 退出前必须完成的事项清单

#### 1. 员工处理（按优先级）
- **提前通知**：根据WARN Act，100人以上公司需提前60天通知大规模裁员
- **最终工资**：必须在最后工作日支付所有应得工资（包括未使用的PTO）
- **COBRA通知**：提供健康保险延续选项（适用于20人以上公司）
- **推荐信**：为表现良好的员工提供推荐信，维护行业关系

#### 2. 资产处理清单
- **数字资产**：
  - 服务器：备份所有数据后关闭实例
  - 域名：续费或转让，避免被抢注
  - SaaS订阅：取消自动续费，导出数据
  - 代码仓库：做好最终备份，设置仓库为只读
- **物理资产**：
  - 办公室：按租约条款处理，避免违约罚款
  - 设备：出售或捐赠，保留收据用于税务
  - 知识产权：确保所有IP归属清晰，便于未来使用

#### 3. 投资人沟通策略
- **透明及时**：在做出最终决定前2周通知主要投资人
- **书面报告**：提供简明的关闭原因、资金使用情况、剩余资产
- **后续支持**：主动帮助投资人进行税务注销（K-1表格处理）
- **声誉维护**：强调学到了什么，而非只强调失败

#### 4. 法律与税务事项
- **公司解散**：提交Articles of Dissolution给州政府
- **税务清算**：提交最终税表（Form 1120 for C-Corp）
- **债权处理**：按优先级偿还债务（员工工资 > 税款 > 商业债务）
- **记录保存**：保留所有公司文件至少7年（IRS要求）
