# 03 失败与退出路径

目标：体面地退场，也是一种本事。别让创业失败变成人生的污点，而是一段“昂贵但牛逼”的经历。

## “我想再撑一个月”后的悲剧

- **“账上只剩 $5000 了，下个月要发 $10000 的工资”**  
  （恭喜你，你已经违法了。拖欠工资在美国是很严重的，劳工局分分钟教你做人。）
- **“F1 OPT 还有 60 天到期，公司黄了”**  
  （别慌，你是去申请 Day 1 CPT，还是赶紧找个婚结了？这是个战术问题。）
- **“投资人说再看看数据就给钱”**  
  （翻译：没戏。赶紧准备 B 计划，不管是去大厂搬砖还是回国接班。）
- **“我要把公司注销了，直接不管就行了吧？”**  
  （Delaware 州政府：你想得美。欠税罚款会一直累积，直到你再次踏入美国海关的那一刻。）

## 等待好心人贡献内容

- **触发停止线（Stop-Loss）**：
  - 现金流警戒线：剩多少钱必须关门？（一般来说是 3 个月运营成本 + 关门费用）。
  - 心理防线：当你开始失眠、脱发、不想见人的时候，该停了。
- **退出前必须完成的事项清单**：
  - 怎么合法地解雇员工（包括你自己）？
  - 怎么处理剩下的服务器、SaaS 订阅和办公室租约？
  - 怎么跟投资人交代（虽然他们可能已经把你拉黑了）？
- **失败复盘与再出发**：
  - 把这次失败写进简历里，是大厂眼里的加分项还是减分项？
  - 下次创业，记得先搞定身份。
