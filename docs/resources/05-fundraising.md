# 05 融资指南

目标：在“饿死”和“卖身”之间找到第三条路。最贵的钱是股权，最好的钱是白送（Non-dilutive）。

## F1 Founder 的融资鄙视链

1.  **Non-dilutive Capital（非稀释性资本）**：
    *   **定义**：拿钱，不给股份。你的控制权 100% 还在手里。
    *   **例子**：
        *   **Grant（政府/机构拨款）**：SBIR/STTR（注意：通常要求绿卡/公民，但部分学校项目有例外）、校友基金无偿资助。
        *   **Competition Prize（大赛奖金）**：各大高校创业大赛（New Venture Competition），奖金几万刀，Do not Dilute。
        *   **Revenue-Based Financing (RBF)**：你有流水，我借你钱，你用未来收入还。适合 SaaS/电商。
    *   **F1 优势**：学校里的奖金通常不看身份，只要你是个好学生（或者好项目）。

2.  **Friendly VC / Angel（友好天使）**：
    *   **定义**：校友、导师、信得过的大佬。
    *   **特点**：看人投，不纠结你的 Visa 还有几天过期。
    *   **F1 策略**：刷脸。你的 .edu 邮箱就是你最好的背书。

3.  **Institutional VC（机构 VC）**：
    *   **定义**：职业投资人。
    *   **特点**：极度理性（冷血）。会问你“创始人被遣返了公司怎么办？”
    *   **F1 策略**：用 Data Traction 说话，或者找一个有绿卡/公民的 Co-founder 站台。

4.  **Predatory VC（掠夺性资本）**：
    *   **定义**：看你身份急，压低估值，签对赌协议。
    *   **特征**：“我可以帮你搞定 O-1，但你要多让 10% 股份。”（快跑！）

## 建议先补充

- **学校创业大赛日历**：哪里有 Non-dilutive 的钱？
- **F1 友好投资人名单**：哪些机构投过国际生？
- **Safe 协议 (YC) 避坑指南**：别把 Cap Table 签烂了。
