# 05 融资指南

目标：在“饿死”和“卖身”之间找到第三条路。最贵的钱是股权，最好的钱是白送（Non-dilutive）。

## F1 Founder 的融资鄙视链

1.  **Non-dilutive Capital（非稀释性资本）**：
    *   **定义**：拿钱，不给股份。你的控制权 100% 还在手里。
    *   **例子**：
        *   **Grant（政府/机构拨款）**：SBIR/STTR（注意：通常要求绿卡/公民，但部分学校项目有例外）、校友基金无偿资助。
        *   **Competition Prize（大赛奖金）**：各大高校创业大赛（New Venture Competition），奖金几万刀，Do not Dilute。
        *   **Revenue-Based Financing (RBF)**：你有流水，我借你钱，你用未来收入还。适合 SaaS/电商。
    *   **F1 优势**：学校里的奖金通常不看身份，只要你是个好学生（或者好项目）。

2.  **Friendly VC / Angel（友好天使）**：
    *   **定义**：校友、导师、信得过的大佬。
    *   **特点**：看人投，不纠结你的 Visa 还有几天过期。
    *   **F1 策略**：刷脸。你的 .edu 邮箱就是你最好的背书。

3.  **Institutional VC（机构 VC）**：
    *   **定义**：职业投资人。
    *   **特点**：极度理性（冷血）。会问你“创始人被遣返了公司怎么办？”
    *   **F1 策略**：用 Data Traction 说话，或者找一个有绿卡/公民的 Co-founder 站台。

4.  **Predatory VC（掠夺性资本）**：
    *   **定义**：看你身份急，压低估值，签对赌协议。
    *   **特征**：“我可以帮你搞定 O-1，但你要多让 10% 股份。”（快跑！）

### 3. SAFE协议避坑指南（YC标准版）

#### 关键条款解析
- **Valuation Cap**：公司估值上限。越低对投资人越有利。
- **Discount Rate**：下一轮融资时的折扣。通常15-25%。
- **MFN Clause**：最惠国条款。确保你获得不低于其他投资人的条件。
- **Pro Rata Rights**：按比例跟投权。可能影响后续融资。

#### F1创始人特别注意
1. **信息权条款**：避免要求定期向投资人汇报运营情况（可能被视为"管理参与"）
2. **董事会观察权**：谨慎给予，可能影响"被动投资者"身份认定
3. **个人担保**：**绝对不要签**！用公司资产担保，而非个人资产
4. **对赌协议**：避免任何与个人身份挂钩的业绩对赌

#### 标准SAFE文件
- [YC SAFE (Simple Agreement for Future Equity)](https://www.ycombinator.com/documents/)
- 推荐使用：**Post-Money SAFE**（更透明，易于计算股权稀释）

### 4. 融资时间线规划（F1 OPT时间窗）

| 时间点 | 融资目标 | 身份状态 | 关键行动 |
|--------|----------|----------|----------|
| **毕业前6个月** | 概念验证 | F1在读 | 参加创业大赛，建立原型 |
| **毕业前3个月** | 天使轮 | F1即将结束 | 申请OPT，寻找天使投资人 |
| **OPT第1-3个月** | Pre-seed | OPT第一年 | 完成首轮$50K-$150K融资 |
| **OPT第4-9个月** | Seed轮 | OPT第一年 | 达到PMF，寻求$500K-$1M |
| **OPT第10-12个月** | 规划A轮 | OPT第一年结束前 | 要么融资成功，要么准备身份转换 |
| **STEM OPT期间** | A轮扩展 | STEM OPT | 规模化扩张，准备后续融资 |

### 5. 投资人沟通策略（针对身份问题）

#### 如何回答"创始人身份问题"
**不要说的**："我的签证还有X个月到期，急需融资解决身份"
**要说的**："我们有明确的身份规划路径，包括O-1/L-1选项，并且有美国合伙人负责日常运营"

#### 准备材料清单
1. **身份规划文档**：清晰的签证转换路径和时间线
2. **美国合伙人背景**：绿卡/公民合伙人的简历和角色说明
3. **业务连续性计划**：如果创始人暂时离境，业务如何继续运营
4. **法律意见书**：律师出具的身份合规意见（可选但有力）
