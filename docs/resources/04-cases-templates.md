# 04 案例与模板

目标：这里有一堆“血淋淋”的教训，还有能帮你省几千刀律师费的模板。

## “如果当初有这个模板就好了”

- **“Case #001：为了省 $500 律师费，最后赔了 $50,000”**  
  （用网上下载的免费合同签外包，结果代码有Bug还无法维权。IP条款没写清，投资人尽职调查时直接给跪了。）
- **“Case #002：创始人吵架，公司瘫痪”**  
  （没有Vesting协议，CTO干了三个月跑了，带走了40%的股份。CEO只能给新招的CTO跪下。）
- **“Case #003：被钓鱼执法”**  
  （在LinkedIn上写“Founder @ Stealth Mode”，结果收到了USCIS的“关怀信”。）

## 真实案例库（匿名处理）

### Case #001：$500律师费 vs $50,000赔偿
**背景**：F1学生用网上下载的免费合同签外包开发，项目完成后代码有严重安全漏洞。
**问题**：合同中没有明确的IP归属条款和保修期，无法追究开发者责任。
**结果**：重新开发花费$50,000，投资人在尽职调查时发现问题后退出。
**教训**：早期合同投资是最高回报的投资。

### Case #002：创始人内斗，公司瘫痪
**背景**：三个F1学生创业，口头约定平分股权，没有vesting协议。
**问题**：CTO三个月后因家庭原因退出，带走33%股权。
**结果**：新CTO要求同等股权，公司股权结构无法支撑后续融资。
**教训**：创始人协议和vesting是创业第一课。

### Case #003：LinkedIn上的“钓鱼执法”
**背景**：F1学生在LinkedIn上写"Founder @ Stealth Startup"，被学校国际办发现。
**问题**：USCIS收到匿名举报，启动身份审查。
**结果**：花费$8,000律师费证明"被动投资者"身份，OPT延期被拒。
**教训**：社交媒体是公开记录，谨慎描述你的角色。

## 实用模板库

### 1. 风险评估自查表
```markdown
# 创业风险评估表

## 身份风险（F1特有）
- [ ] 是否有美国公民/绿卡合伙人？
- [ ] 是否签署了Passive Investor Agreement？
- [ ] 是否在公司文件中避免出现Officer头衔？
- [ ] 是否有完整的身份时间线记录？

## 法律风险
- [ ] 公司注册文件是否齐全？
- [ ] 创始人协议是否签署？
- [ ] 员工/承包商合同是否有IP条款？
- [ ] 是否了解所在州的劳动法？

## 财务风险
- [ ] 是否有3个月运营现金储备？
- [ ] 是否设置了工资支付预警线？
- [ ] 是否了解公司税务义务？
- [ ] 个人资产与公司资产是否隔离？

## 评分
- 0-3个✅：高风险，立即暂停行动
- 4-7个✅：中等风险，需要专业咨询
- 8-10个✅：相对安全，可以继续
```

### 2. 律师咨询准备模板
```markdown
# 律师咨询准备材料

## 个人背景
- 签证类型：F1
- 当前状态：OPT (STEM Extension)
- 毕业时间：YYYY-MM
- 专业：计算机科学

## 公司情况
- 公司名称：[Company Name]
- 注册州：Delaware
- 公司类型：C-Corp
- 注册时间：YYYY-MM

## 我的角色
- 股权比例：20%
- 头衔：无（避免Officer/Director头衔）
- 实际参与：仅提供技术建议（每月<5小时）
- 收入：无工资，仅股权

## 具体问题
1. 作为F1持有20%股权但不参与运营，需要哪些文件证明"被动投资者"身份？
2. 如果公司获得融资，我的股权变更是否需要向USCIS报告？
3. 公司盈利分红给我，税务上如何处理？
4. 如果未来申请O-1签证，这段创业经历如何呈现？

## 期望产出
- Passive Investor Agreement草案
- 股东会议记录模板
- USCIS沟通话术建议
```
