# 06 招人指南

目标：招聘是创始人最重要的产品。别找大爷，找战友（和干活的）。

## 没人想给 F1 Founder 打工（除非你真的牛逼）

### 真实痛点

1.  **“我们是正规军，不是草台班子”**
    *   **问题**：没绿卡，没 H1B Sponsor 资质，大牛不理你。
    *   **解法**：用愿景（Vision）画饼，用期权（Option）锁人，或者找 **Day 1 CPT / OPT** 的难兄难弟。
    *   **警告**：别为了招人乱承诺办身份，做不到就是欺诈（诈骗人才）。

2.  **“没钱怎么招大神？”**
    *   **问题**：没钱。
    *   **解法**：找 **Intern（实习生）**。
        *   **CPT Intern**：学校允许兼职，工资低（甚至 Unpaid），只要你给学分证明。
        *   **OPT Intern**：刚毕业想找全职但找不到，给你当几个月便宜劳动力（甚至 Volunteer）。
    *   **警告**：Unpaid Intern（无薪实习）必须符合劳工法（主要受益人必须是实习生，不是公司）。别把人当免费全职员工用，会被告。

3.  **“Co-founder 怎么找？”**
    *   **问题**：你也 F1，我也 F1，俩难兄难弟抱团取暖？
    *   **解法**：找互补（你技术，他 Sales；你 F1，他绿卡/公民）。
    *   **警告**：**Vesting（股权兑现）必须签！** 必须签！必须签！如果不签，他干三个月跑了带走 50% 股份，公司直接猝死。

4.  **“小心被‘志愿者’举报”**
    *   **问题**：有些“志愿者”干活不顺心，回头告你非法用工。
    *   **解法**：Volunteer Agreement 签好。明确没有雇佣关系，没有工资期待。

## 实习生招聘合规指南

### Unpaid Intern 合法性测试（DOL 7项测试）
根据美国劳工部（DOL）规定，无薪实习生必须满足**全部7项标准**：

1. 实习类似教育环境中的培训
2. 为实习生提供培训
3. 不取代正式员工
4. 雇主不从中获得直接利益
5. 实习生不一定有工作保障
6. 双方理解这不是有偿工作
7. 实习期结束后不一定有工作机会

**关键红线**：
- 不能将实习生当作免费劳动力
- 必须提供教育价值
- 不能替代正式员工工作
- 公司不能从实习生工作中获得直接经济利益

### 实习生分类与合规要求

| 实习类型 | 薪资要求 | 法律要求 | 适合F1学生 |
|----------|----------|----------|------------|
| **Unpaid Intern** | 无薪，但可提供津贴 | 必须满足DOL 7项测试 | 需要CPT/OPT授权 |
| **Academic Credit** | 无薪，换取学分 | 学校合作项目 | 需要学校批准 |
| **Paid Intern** | 至少最低工资 | 遵守FLSA规定 | CPT/OPT均可 |
| **Co-op Program** | 带薪实习 | 学校合作项目 | 需要学校批准 |

## Co-founder协议核心条款

### 股权分配与Vesting
```markdown
# 创始人股权分配模板

## 股权分配
- 创始人A（技术）：40% (4年vesting，1年cliff)
- 创始人B（业务）：40% (4年vesting，1年cliff)  
- 期权池：20% (为员工预留)

## Vesting计划
- 标准：4年vesting，1年cliff
- 加速条款：控制权变更时加速vesting
- 回购权：公司有权以成本价回购离职创始人股份

## 决策机制
- 日常决策：简单多数
- 重大决策（融资、出售等）：2/3多数
- 僵局解决：指定独立董事
```

### 创始人离职处理
1. **友好离职**：未vest部分收回，已vest部分保留
2. **因故离职**：公司有权以成本价回购股份
3. **因故解雇**：公司可折价回购股份

## 早期员工期权设计

### 期权池规模
- 种子轮：10-15% 总股本
- A轮前：15-20% 总股本
- 关键员工：1-5% 个人期权（4年vesting）
- 早期员工：0.1-1% 不等

### 期权分配策略
```markdown
# 早期员工期权分配指南

## 关键角色参考
- CTO/技术负责人：1-5%
- 早期工程师：0.5-2%
- 产品负责人：1-3%
- 市场/销售负责人：0.5-2%

## Vesting计划
- 标准：4年vesting，1年cliff
- 加速条款：控制权变更时50%加速
- 行权价：按409A估值设定
```

## F1/OPT员工特殊考虑

### CPT实习合规
1. **CPT实习要求**：
   - 必须与专业相关
   - 需要学校DSO批准
   - 每周工作限制（通常20小时/学期）
   - 必须获得学分或学术要求

2. **OPT实习要求**：
   - STEM OPT需要E-Verify雇主
   - 必须与专业直接相关
   - 每周至少工作20小时
   - 定期向学校报告

### 国际员工招聘流程
1. **招聘阶段**：明确说明无法赞助H-1B
2. **OPT阶段**：利用OPT/STEM OPT（最长3年）
3. **长期规划**：O-1、L-1、EB-1/2/3绿卡路径
4. **风险披露**：明确说明无法保证签证支持

## 薪酬结构建议

### 早期阶段薪酬结构
| 职位级别 | 现金:股权比例 | 基本薪资范围 | 股权范围 |
|----------|---------------|--------------|----------|
| 创始人   | 低现金+高股权 | 市场价50-70% | 5-20%    |
| 早期员工 | 市场价70-80%  | 市场价80-90%  | 0.5-5%   |
| 关键员工 | 市场价80-90%  | 市场价90-100% | 0.1-2%   |

### 股权激励时间表
- **第1年**：25% vesting（1年cliff后）
- **第2年**：每月vest 2.08%（剩余3年）
- **离职处理**：90天内行权，否则失效
- **行权价**：按409A估值设定

## 风险控制

### 法律风险
1. **错误分类**：将员工错误分类为contractor
2. **工资违规**：未付加班费、最低工资违规
3. **歧视风险**：招聘中的歧视性做法
4. **I-9违规**：未正确验证工作资格
