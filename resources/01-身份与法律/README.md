# 01 身份与法律

目标：在“可做”和“不能做”之间建立清晰边界，避免还没融到资，人先没了（物理遣返）。

## 经典“作死”场景大赏

- **“我就是帮朋友改个 Bug，没收钱”**  
  （USCIS：你提供了劳务，你产生了价值，你就是非法打工。解释信准备好了吗？）
- **“我只拿了点股份，没参与运营”**  
  （如果你每天在 Slack 上指点江山，那叫“积极管理”。被举报的时候，聊天记录全是呈堂证供。）
- **“我在国内远程创业，钱打给爸妈”**  
  （只要人在美国境内敲代码，就是在美国工作。别以为 IRS 查不到支付宝。）
- **“用 CPT 开公司，学校说没问题”**  
  （Day 1 CPT 是你的救命稻草，也可能是你的不定时炸弹。RFE 来了，你学校的国际办能帮你挡子弹吗？）

- **F1 常见创业场景边界**：
  - “只出钱不出力”的正确姿势
  - “无薪实习”到底是不是坑？

- **咨询律师前材料清单**：

  - 省钱攻略：先把你的 Story 整理成文档，别按小时给律师讲相声。
- **关键沟通记录模板**：
