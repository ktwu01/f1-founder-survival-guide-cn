# Project Scratchpad

## Background and Motivation

用户希望创建一个面向中文 F1 创业者的“进攻 + 退路”资源仓库，先搭建非常简单的中文骨架，后续再补充复杂上下文与深度规划。

## Key Challenges and Analysis

- 需要保持内容简单可扩展，避免一开始过度设计。
- 需要明确“信息整理”定位，避免被误解为法律建议。
- 需要保留英文高质量来源，方便交叉验证。
